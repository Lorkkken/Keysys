This is my personal project, Keysys.

It simulates opening cases like in CS2.

Its features include:

1) Registration and login
2) Opening cases
3) Inventory with preferences
4) Full account date saving in Google FireBase

If the program doesn't open for you, the problem is most likely in the libraries. Download them in the terminal:

pip install pygame requests

I'll likely continue to develop it, but not as quickly as my previous mini-projects.
